# 🚀 Portfolio — Abraham Ogbu

A modern full-stack developer portfolio showcasing projects, skills, and experience. Built with Next.js for the frontend and integrated backend services using Node.js, MongoDB, and Nodemailer for email functionality.

---

## 🌐 Live Demo

👉 https://portfolio-2-azure-delta.vercel.app

---

## 🧠 Features

- ⚡ Modern responsive UI
- 📂 Projects showcase with live demos and GitHub links
- 🧾 Resume preview and download
- 📬 Contact form with email functionality (Nodemailer)
- 🌙 Clean and minimal design

---

- Nodemailer (Email handling)

- Git & GitHub

## ⚙️ Environment Variables

Create a `.env.local` file in the root and add:

```
MONGODB_URI=your_mongodb_connection_string
EMAIL_USER=your_email
EMAIL_PASS=your_email_password
```

---

## 🧪 Getting Started

### 1. Clone the repository

```
git clone https://github.com/Abraham3stack/portfolio.git
cd portfolio
```

### 2. Install dependencies

```
npm install
```

### 3. Run development server

```
npm run dev
```

Open http://localhost:3000 to view the app.

---

## 📌 Project Structure

```
/app        → Next.js app directory
/components → Reusable UI components
/sections   → Page sections (Hero, Projects, Contact, etc.)
/public     → Static assets (images, resume)
```
